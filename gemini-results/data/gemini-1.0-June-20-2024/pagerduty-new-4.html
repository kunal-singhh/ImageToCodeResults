PagerDuty Incident Viewer

The PagerDuty Incident Viewer is a tool that allows you to view a list of your current PagerDuty incidents.

PagerDuty Incidents

Incident ID Title Status Actions

Actions:

  • View Incident: This action will open the incident in a new window.
  • Acknowledge Incident: This action will acknowledge the incident.
  • Resolve Incident: This action will resolve the incident.
